# wwtbam
При запуске приложения требуется выбрать файл с вопросами и вариантами ответов. Расширение файла - txt. 
Содержимое файла должно выглядеть следующим образом: 
```
1;Какая раса у Сильваны Ветрокрылой?;Эльф крови;Ночной эльф;Отрёкшаяся;Эльф бездны;А
2;Какое прозвище у Гарроша?;Адский рык;Адский крик;Адский рёв;Адский ор;B
```

Маска заполнения файла: 
```
Раунд;Вопрос;Вариант A;Вариант B;Вариант C;Вариант D;Правильный ответ
```

Допускается содержимое файла с хедерами.
```
Раунд;Вопрос;Вариант A;Вариант B;Вариант C;Вариант D;Правильный ответ
1;Какая раса у Сильваны Ветрокрылой?;Эльф крови;Ночной эльф;Отрёкшаяся;Эльф бездны;А
2;Какое прозвище у Гарроша?;Адский рык;Адский крик;Адский рёв;Адский ор;B
```

Допускается создание файла непосредственно из интерфейса приложения.
